# Product Overview

AccessFlow is a web application designed to help disabled and neurodivergent job seekers create better job applications with minimal friction and maximum empathy.

## Core Principles

- Accessibility-first design (WCAG-aware UI)
- Privacy-by-default (no unnecessary PII retention)
- Empathetic user experience tailored for neurodivergent users
- Low friction application process

## Target Users

Disabled and neurodivergent individuals seeking employment opportunities.
