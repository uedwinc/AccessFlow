# Technology Stack

## Frontend
- React
- Node.js runtime

## Backend & Infrastructure
- AWS Lambda (serverless functions)
- API Gateway (REST API endpoints)
- DynamoDB (NoSQL database)
- S3 (object storage)
- CloudFront (CDN)
- Amazon Bedrock (AI/ML services)

## Architecture Pattern
Serverless architecture using AWS managed services.

## Non-Functional Requirements
- Free-Tier-optimized: Design with AWS Free Tier limits in mind
- Accessibility: All UI components must be WCAG-aware
- Privacy: Minimize PII retention, privacy-by-default approach
